# Task A1: Create a File

## Objective
Create a file named `hello.txt` in the current directory with the exact content:
```
Hello World
```

## Requirements
1. The file MUST be named exactly `hello.txt`
2. The content MUST be exactly `Hello World` (no trailing newline required, but acceptable)
3. Use the `write` tool or `bash` with echo/cat to create the file

## Deliverables
- `hello.txt` - the created file

## Success Criteria
- File `hello.txt` exists
- File contains the text "Hello World"

## DO NOT
- Create any other files
- Add extra content
- Rename or move the file
