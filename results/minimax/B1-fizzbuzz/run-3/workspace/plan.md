# Task B1: FizzBuzz Implementation

## Objective
Create a Python file `fizzbuzz.py` that implements the classic FizzBuzz algorithm.

## Requirements
1. Create a file named `fizzbuzz.py`
2. Implement a function `fizzbuzz(n)` that:
   - Takes an integer `n` as input
   - Returns a list of strings for numbers 1 to n where:
     - Numbers divisible by 3 AND 5 → "FizzBuzz"
     - Numbers divisible by only 3 → "Fizz"
     - Numbers divisible by only 5 → "Buzz"
     - Other numbers → the number as a string
3. Include a `if __name__ == "__main__":` block that prints `fizzbuzz(15)`

## Expected Output
When running `python fizzbuzz.py`, it should print:
```
['1', '2', 'Fizz', '4', 'Buzz', 'Fizz', '7', '8', 'Fizz', 'Buzz', '11', 'Fizz', '13', '14', 'FizzBuzz']
```

## Deliverables
- `fizzbuzz.py` - The Python implementation

## Success Criteria
- File exists and is valid Python (no syntax errors)
- Function `fizzbuzz(15)` returns correct output
- Code runs without errors

## DO NOT
- Use external libraries (only built-in Python)
- Create additional files
- Include test frameworks
